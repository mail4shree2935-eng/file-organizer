# file-organizer
📂 File Organizer (Python)

A simple Python script that automatically organizes files in a folder into categories like Images, Documents, Videos, and more.

---

🚀 Features

- 📁 Automatically sorts files into folders
- 🖼️ Supports images, documents, videos, audio, etc.
- ⚡ Fast and easy to use
- 🧹 Keeps your workspace clean and organized

---

🛠️ Tech Stack

- Python 🐍
- OS & Shutil modules

---

📂 Project Structure

file-organizer/
│── organizer.py
│── README.md

---

▶️ How to Run

1. Clone the repository:

git clone https://github.com/your-username/file-organizer.git

2. Navigate to the folder:

cd file-organizer

3. Run the script:

python organizer.py

---

🎯 How It Works

- The script scans a folder
- Identifies file types based on extensions
- Moves them into respective folders

---

📌 Example Categories

- Images → .jpg, .png
- Documents → .pdf, .docx
- Videos → .mp4, .mkv
- Audio → .mp3
